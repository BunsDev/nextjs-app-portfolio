- Framework: Next.js
- Library: React
- Language: TypeScript
- UI Library: Material UI
- Styling: Tailwind CSS
- Deployment: Github Pages

## Running Locally

This application requires Node.js v18.17+.

```bash
<clone>
cd <clone folder>
yarn install
yarn build
yarn dev
```
