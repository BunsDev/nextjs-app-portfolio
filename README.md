# react-portfolio-app

- Framework: Next.js
- Library: React
- Language: TypeScript
- UI Library: Material UI
- Styling: Tailwind CSS and Emotion Styled Components
- Deployment: tbd

## Running Locally

This application requires Node.js v18.17+.

```bash
git clone https://github.com/ellamaolson/react-portfolio-app.git
cd react-portfolio-app
yarn install
yarn build
yarn dev
```
